Sobre eu:

Entrei no mundo da programação por volta dos 11 anos (tenho 15 atualmente), com a ideia de criar projetos com arduino. Depois comecei a aprender C e C++, comecei a participar de competições de programação e a desenvolver projetos maiores. Apartir dai fui tomando paixão pela area pelas inumeras possibilidades de desenvolver coisas novas. Principais Linguagens e Frameworks que utilizo atualmente:

  • HTML
  • CSS
  • Java-Script
  • Python
  • ++ C++
  • Django
  • Opencv
  • Arduino
  • MySQL
  • PostgreSQL
  • Todo meu conhecimento veio de Livros, documentação, cursos e principalmente pela pratica.
  • Comecei desenvolver para Web em 2019 com PHP e atualmente utilizo Python-Django.
  • Gosto de desenvolver projetos e colocar no Github.
  • Para mim linguagens são apenas ferramentas, onde a melhor e a que supli suas nessecidades.
